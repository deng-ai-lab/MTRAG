# Evaluating MTRAG 🔍

Run the following command to evaluate the MTRAG model on our proposed MTR-Bench, which is a benchmark specifically designed to assess multi-region referring understanding.

Inference:
```bash
bash eval/mtr-bench/mtr_bench_infer.sh
```
Evaluation:
```bash
bash eval/mtr-bench/multi_region_gpt_eval.sh
```
